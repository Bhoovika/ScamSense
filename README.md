# ScamSense

Scam and cybercrime losses in Singapore reached $913.1 million in 2025, even though there was a decrease in the number of cases from the previous year, with scammers increasingly operating across the multiple languages spoken in Singapore - English, Singlish, Malay, Tamil, and Mandarin (Singapore Police Force, 2026). The current automated scam detectors are mostly monolingual models trained on English language datasets. This project presents ScamSense, an end-to-end multilingual scam detection system designed to address this gap. A combined dataset of approximately 135,000 messages was constructed from real-world sources (phishing emails, SMS spam, fraudulent job postings) and synthetically generated scam messages belonging to the scam categories outlined in the scam taxonomy of Singapore Police Force (SPF) for 2025, translated into Malay, Tamil, and Mandarin using the NLLB-200 neural machine translation model. A transformer-based classifier was fine-tuned on this dataset using XLM-RoBERTa, achieving a weighted F1-score of 0.993 and an AUC of 0.999, beating a multilingual BERT baseline. The classifier was integrated into a three-agent LangGraph pipeline combining SHAP-based token-level explainability with a retrieval-augmented generation component grounded in official SPF 2025 advisory data, producing risk-scored, evidence-backed explanations for each prediction. The complete system was exposed through a FastAPI backend with persistent logging, built entirely on free-tier infrastructure. Evaluation revealed strong overall performance alongside specific multilingual limitations, particularly in detecting impersonation scams written in formal Mandarin, which are discussed as priority directions for future work.
